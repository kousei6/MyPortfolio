概要

Pygameを使用してPacmanを実装した開発チームの経験共有します。５人のチームでGitHubを活用し、 協力しながら開発を進めました。プロジェクトの環境のWindows11上のAnaconda仮想環境で構築し、Python3.10以上とPygame2.5.2を 使用しました。チーム開発では、GitHubのリポジトリを利用し、ブランチ戦略やプルリクエストを活用しながらスムーズに進行しました。

game screen

ゲーム画面

Pacmanの基本的なルールを再現し、プレイヤーは矢印キーで移動できます。マップ上には通常エサやパワーエサが配置されており、パワーエサを食べることで一定時間 ゴーストを倒せます。ゴーストは通常時に接触するとゲームオーバーになるが、異なる動きをする複数のモードを持っています。また、ワープトンネルを活用して特定の地点間を移動できる機能も実装しました。

source code

ソースコード

本ゲームの実装には,pythonの機能を活用して以下のようなクラスを設計しました。
・Mapクラス:マップ情報を管理(マップデータ、エサの位置、ワープトンネルなど)
・Playerクラス:Pacmanの移動やエサの取得処理
・Ghostクラス:ゴーストの行動パターンや状態管理
・GameManagerクラス:ゲーム全体の制御(開始、終了、スコア管理)
また、スムーズな開発のため、GitHubを用いてブランチ戦略を採用し、各機能ごとに開発を進めました。
